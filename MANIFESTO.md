# MANIFESTO — salOWN

**salOWN** is a SaaS platform for multi-salon management. Starting point: a custom system for I CUT Whitecross Barbers.
Now an independent platform competing with Booksy, Fresha, Treatwell.

## Core Philosophy: Grabbing

salOWN doesn't try to replace existing channels — it unifies them.
A booking from Booksy, a booking from Treatwell, a walk-in — all in one system.
The salon owner doesn't have to abandon Booksy; salOWN is built on top.

## Goal

- `salown.com` → Consumer marketplace (like Booksy/Fresha) — salon discovery, profile, online booking
- `hub.salown.com` → **salOWNHub** — partner/business portal (like Fresha Partner, Treatwell Connect)
- `noreply@salown.com` → All platform transactional emails (via Brevo)

## Whitecross connection

Whitecross is salOWN's **premium pilot tenant**. Every new feature is tested there first.
The developer (Alfa) is also the owner of Whitecross — this gives both the developer and the user perspective.
